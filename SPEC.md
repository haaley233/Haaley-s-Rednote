# 何意味·内容工作台 SPEC

## 一句话定位
帮助大学生创作者管理选题、记录爆款、可视化数据的一站式工具

## 功能模块

### 1. 选题灵感库
- 输入关键词 → 生成10个标题方向 + Hook句式
- 本地保存历史选题到 localStorage
- 复制标题到剪贴板

### 2. 内容日历
- 记录发过的笔记：日期、标题、曝光、点赞、收藏、涨粉
- 自动生成折线图展示成长曲线（用 Chart.js）
- 新增/编辑/删除笔记记录

### 3. 竞品素材库
- 收藏爆款笔记：链接、标题、数据（点赞/收藏/评论）、爆款原因标签
- 按标签筛选
- 标记"已研究/待研究"

### 4. 个人名片
- 展示小红书账号数据（粉丝/获赞收藏/单篇最高曝光）
- 一键复制数据为文本格式

## 设计规范

### 色彩
- 背景：#1A1A2E（深夜蓝黑）
- 卡片：#16213E（深蓝灰）
- 强调色：#FF6B6B（珊瑚红）
- 辅助色：#FEC89A（暖橙）
- 文字主色：#FFFFFF
- 文字次色：#A0A0B0

### 字体
- 主字体：思源黑体（Noto Sans SC）/ sans-serif
- 数字/英文：Inter

### 布局
- 顶部导航固定，高度 64px
- 主内容区最大宽度 1200px，居中
- 卡片圆角 12px，间距 20px
- 过渡动画 0.3s ease

## 技术栈
- 纯前端：HTML + CSS + Vanilla JS
- 图表：Chart.js（CDN）
- 图标：Lucide Icons（CDN）
- 数据：localStorage（无后端）
- 部署：Vercel（静态）

## 部署
1. GitHub 新建仓库
2. Vercel 导入仓库
3. 自动部署，无需配置
